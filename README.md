# Car Lot Project Portfolio

A Java project that models a simple car lot system. This repository is intended as a portfolio piece demonstrating object-oriented programming concepts.

## Features

- Add and remove cars from the lot
- View available inventory
- Search/filter cars (e.g., by make/model/year)

## Tech Stack

- Java

## Getting Started

### Prerequisites

- Java 17+ (or your installed Java version)

### Run

1. Clone the repo
2. Open the project in your IDE (IntelliJ IDEA / Eclipse)
3. Build and run the main class

## Project Structure

- `src/` – Java source code
